01

Prerequisites

Before using the extension, confirm the following are in place:

  • Google Chrome installed (not Chromium, Edge, or other browsers)
  • Extension installed and enabled in Chrome at chrome://extensions
  • Network access to the AC Smart 5 controller — same local network or VPN
  • The device IP address (default: 172.16.0.140)
  • LG AC Smart 5 username and password
02

First-Time Setup: Accept the Device Certificate

The AC Smart 5 uses a self-signed HTTPS certificate. Chrome blocks it until you manually accept it — this is a one-time step per Chrome profile.

  1. Open a new Chrome tab
  2. Navigate to https://<device-ip>/ — for example, https://172.16.0.140/
  3. Click Advanced → Proceed to 172.16.0.140 (unsafe)
  4. The LG AC Smart 5 login page appears — you may close this tab afterward
⚠

If the extension shows a connection error, repeat this step. Self-signed certificates occasionally reset after device reboots or firmware updates.

03

Opening the Side Panel

  1. Click the LG Thermostat icon in the Chrome toolbar
  2. The side panel opens on the right side of the current tab
  3. The panel is scoped to that tab — switching to another tab hides it
  4. Return to the original tab to bring the panel back
ⓘ

The side panel intentionally follows a single tab. To use it on a different tab, click the toolbar icon again from that tab.

04

Configuring Settings

Click the gear icon in the panel header to open Settings. All settings persist across sessions — you only need to enter them once.

Field Description
Device IP Address IP of the AC Smart 5 controller (e.g., 172.16.0.140)
Username AC Smart 5 login username
Password AC Smart 5 login password
Poll interval (minutes) How often live temperature data refreshes. Default: 1 min. Minimum: 0.1 min.
Room data ingest Toggle for supplementary room occupancy data. Off by default.
05

Connecting to the Controller

Automatic connection (credentials saved)

If username and password are entered, clicking Connect will:

  1. Check for an open device tab
  2. If no tab exists, automatically open https://<ip>/ in a background tab
  3. Authenticate using the saved credentials
  4. Load all indoor unit data
  5. Begin live polling on the configured interval

Manual connection (no credentials)

  1. The "Device tab not open" banner appears
  2. Click Open — a new tab opens to the device login page
  3. Log in manually in that tab
  4. Return to the panel and click Connect again

Status indicators

Connected Polling active, all units loaded
Disconnected Not connected to device
Error banner See error message — check IP, credentials, or network
Device tab not open Device tab was closed; reopen manually or re-enter credentials
06

Viewing and Searching Rooms

Once connected, all indoor units are listed as cards. The search bar appears above the room list and receives focus automatically.

Search behavior

  • Type digits to filter by room number — letters are ignored
  • The first digit matches the floor: typing 4 shows all rooms on floor 4
  • Typing 402 narrows to room 402 specifically
  • Clear the field to show all rooms

Room card elements

Element Description
Room name Unit identifier (e.g., 402)
Floor label Floor name from the device (e.g., 4TH FLOOR)
Room temp Current measured temperature in °F
Set temp Target setpoint in °F — use + / − buttons to adjust
Power button Toggle the unit on or off
Mode row COOL  ·  HEAT  ·  FAN  ·  AUTO  ·  DRY
Fan row LOW  ·  MED  ·  HIGH  ·  AUTO
07

Controlling a Room

Adjust setpoint temperature

  • Click + to raise the setpoint by 0.5°C (displayed in °F)
  • Click − to lower the setpoint by 0.5°C
  • The device enforces per-unit min/max limits — typically 16–30°C (61–86°F)
  • Changes are sent immediately; no save button is required

Turn a unit on or off

  • Click the power button (circle icon) in the top-right of the card
  • Active units show the button highlighted
  • The change takes effect immediately

Change mode

  • Click COOL, HEAT, FAN, AUTO, or DRY
  • The active mode is highlighted in blue

Change fan speed

  • Click LOW, MED, HIGH, or AUTO
  • The active speed is highlighted in blue
08

Live Data Refresh

  • The panel polls the device every N minutes per your poll interval setting
  • While the panel is open, it also refreshes every 10 seconds in the foreground
  • The device tab must remain open for polling to continue
⚠

If the device tab is closed while connected, the panel will display: "Device tab was closed. Reopen it to reconnect." Open a tab to the device URL and click Connect to resume.

09

Disconnecting

Open the gear icon to reveal Settings, then click Disconnect.

  • Clears the active connection and stops polling
  • Room list is cleared from view
  • Credentials and IP remain saved for the next session
10

Troubleshooting

Problem Resolution
"Device tab was closed" Open a tab to https://<ip>/ and click Connect
Login failed (HTTP 401 / 403) Verify username and password in Settings
"Tab took too long to load" Accept the device certificate first (see Section 02)
Temperatures not updating Check poll interval; confirm the device tab is still open
No rooms appear after connect Device returned 0 indoor units — verify device IP and network access
Panel disappears when switching tabs Expected behavior — the panel is scoped to the originating tab
Extension not responding Go to chrome://extensions, find LG Thermostat Controller, and click the reload icon
11

Data Notes

  • All temperatures are displayed in Fahrenheit
  • API communication with the device uses Celsius internally
  • Room occupancy data is fetched once at connect time — not on every poll cycle
  • Credentials are stored in Chrome's local extension storage and are not synced across devices or profiles
